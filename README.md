# 🤖 AI Resume Maker

## 📌 Overview
AI Resume Maker is a smart application that generates professional resumes using Natural Language Processing (NLP). It takes user input and automatically creates well-structured resume content.

## 🚀 Features
- Generate resume using AI
- Convert user input into professional content
- Supports multiple sections (Education, Skills, Experience)
- Fast and automated resume creation
- Clean and user-friendly interface

## 🧠 Technologies Used
- Python
- Flask
- Sentence Transformers
- Transformers (Hugging Face)
- NLP (Natural Language Processing)

## ⚙️ How It Works
1. User enters details (skills, experience, education)
2. AI processes input using NLP models
3. Generates structured resume content
4. Displays or downloads the resume
